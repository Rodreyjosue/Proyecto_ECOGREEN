#AutoGreen – Smart Greenhouse Control System

**AutoGreen** es un sistema automatizado basado en Arduino diseñado para monitorear y controlar variables ambientales clave en cultivos o espacios controlados, optimizando el cuidado de plantas de forma inteligente.

***

## Funcionalidades principales

* **Control de temperatura**  
  Activación automática de ventiladores cuando la temperatura es alta.

* **Sistema de alerta / calefacción**  
  Activación de buzzer o indicador cuando la temperatura baja.

* **Riego automático**  
  Uso de sensor de humedad de suelo para activar una bomba de agua.

* **Control de iluminación**  
  Sensor LDR que activa un LED cuando hay baja luz.

* **Interfaz en tiempo real**  
  Pantalla LCD que muestra estado del sistema y eventos (ej: “Ventilador Activo”).

***

## ⚙️ Tecnologías utilizadas

* **Arduino (C/C++)**
* Sensores:
  * DHT (temperatura)
  * Humedad de suelo
  * LDR (luz)
* Actuadores:
  * Ventiladores
  * Bomba de agua (controlada por relay)
  * LED
  * Buzzer
* LCD 16x2 con I2C

***

## Lógica del sistema

El sistema opera de forma autónoma tomando decisiones en tiempo real:

* Alta temperatura → activa ventilación
* Baja temperatura → genera alerta térmica
* Suelo seco → activa riego automático
* Baja iluminación → enciende luz
